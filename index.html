Sr. UI Developer Assignment


Please follow the following guidelines when building:

1. Build and integrate the sidebar on a web page.
2. The rest of the webpage can be empty.
3. The look and feel of the sidebar has to be exactly the same as in the video. It needs to be pixel perfect. Including the transitions and animations.
4. Also think about responsiveness of the sidebar.
5. Text and icons can change. You can also reduce the number of items in the sidebar in the second column.
6. You are free to use any icon library.
7. Using any CSS, JS framework or library is not allowed. The code has to be written in vanilla css, html5 and pure javascript or typescript. Using any helper library is not allowed.


Bonus:

Implement a dark/Light theme switcher and change colors of the sidebar accordingly.


Submission Guidelines:

1. Please upload the project to github.
2. Please keep the project public to avoid any access related issues.
3. Do Not upload a zip. Please upload the full code.
